# CLEOPATRA Slots Research Workflow

## Overview

Systematic workflow for reverse engineering CLEOPATRA slots from YouTube videos of real Las Vegas slot machines. This research follows a heartbeat-driven approach with 30-minute progress cycles.

## Workflow Steps

### Step 1: Video Shortlist Creation

**Objective:** Build a curated shortlist of high-quality CLEOPATRA gameplay videos.

**Actions:**
- [ ] Search YouTube for "CLEOPATRA slots Las Vegas" and related terms
- [ ] Search for "CLEOPATRA slots real casino" gameplay
- [ ] Search for "CLEOPATRA slots high limit" for detailed views
- [ ] Evaluate videos based on quality criteria:
  - Clear visibility of reels and symbols
  - Multiple complete spin sequences shown
  - Bonus features demonstrated
  - Good video and audio quality
  - Real casino machine (not mobile/online versions)
- [ ] Select exactly 5 best videos for the shortlist
- [ ] Document each video with:
  - Video title
  - YouTube URL
  - Duration
  - Brief description (1-2 sentences)
  - Key strengths (why it's on the shortlist)
- [ ] Prepare shortlist message for Ron

**Deliverable:** List of 5 CLEOPATRA videos ready to send to Ron.

---

### Step 2: User Selection via Telegram

**Objective:** Get Ron to select the video for detailed analysis.

**Actions:**
- [ ] Compose Telegram message with:
  - Introduction explaining the shortlist
  - Numbered list (1-5) of videos with titles and links
  - Brief description of each video's strengths
  - Clear request: "Please reply with a number (1-5) to select which video I should analyze"
- [ ] Send message to Ron via Telegram
- [ ] Wait for Ron's response
- [ ] Record Ron's selection in MEMORY.md
- [ ] Proceed to analysis with the selected video

**Deliverable:** Ron's video selection confirmed and ready for analysis.

---

### Step 3: Video Analysis - Reel Configuration

**Objective:** Determine the technical reel structure.

**Actions:**
- [ ] Watch selected video multiple times
- [ ] Identify and document:
  - Number of reels (typically 5)
  - Number of rows (typically 3)
  - Total visible positions
  - Reel spin behavior and timing
  - Any special reel features (expanding reels, etc.)
- [ ] Take screenshots of reel layout
- [ ] Note any unique reel mechanics
- [ ] Document findings in MEMORY.md

**Deliverable:** Complete reel configuration documented.

---

### Step 4: Video Analysis - Symbol Inventory

**Objective:** Catalog all unique symbols in the game.

**Actions:**
- [ ] Watch video and identify all unique symbols
- [ ] Categorize symbols:
  - **High-Value Symbols:** (typically themed)
    - Cleopatra herself
    - Pharaoh/Egyptian rulers
    - Egyptian artifacts
    - Special themed symbols
  - **Low-Value Symbols:** (typically card ranks)
    - A, K, Q, J, 10, 9 (note which are present)
  - **Wild Symbols:**
    - Appearance and behavior
    - Substitution rules
  - **Scatter Symbols:**
    - Appearance and behavior
    - Bonus triggers
  - **Special Symbols:**
    - Any unique or bonus-related symbols
- [ ] For each symbol, document:
  - Visual description
  - Symbol category
  - Special properties or behaviors
  - Approximate frequency on reels
- [ ] Capture screenshots of each unique symbol
- [ ] Update MEMORY.md with complete symbol inventory

**Deliverable:** Complete symbol inventory with descriptions.

---

### Step 5: Video Analysis - Base Game Mechanics

**Objective:** Understand core gameplay mechanics.

**Actions:**
- [ ] Analyze and document:
  - **Payline Structure:**
    - Number of paylines
    - Payline patterns (if visible)
    - Win evaluation method
  - **Win Conditions:**
    - Minimum symbols for wins
    - Left-to-right or both-ways pays
    - Symbol value hierarchy
  - **Wild Symbol Behavior:**
    - Which symbols wilds substitute for
    - Wild multipliers (if any)
    - Wild stacking or expansion
  - **Base Game Features:**
    - Random wilds
    - Symbol transformations
    - Cascading/tumbling reels
    - Any other special base game mechanics
- [ ] Watch multiple spin sequences to confirm patterns
- [ ] Note win examples and calculations
- [ ] Document all findings in MEMORY.md

**Deliverable:** Complete base game mechanics documentation.

---

### Step 6: Video Analysis - Bonus Features

**Objective:** Understand bonus games and special features.

**Actions:**
- [ ] Identify all bonus features:
  - **Free Spins:**
    - Trigger conditions (e.g., 3+ scatters)
    - Number of free spins awarded
    - Special features during free spins
    - Re-trigger possibilities
  - **Bonus Games:**
    - Pick-em bonuses
    - Wheel bonuses
    - Progressive jackpots
    - Any other bonus rounds
  - **Special Features:**
    - Multipliers (when/how they apply)
    - Symbol upgrades
    - Expanding symbols
    - Mystery symbols
- [ ] For each feature, document:
  - Trigger conditions
  - Feature behavior
  - Potential payouts
  - Any special rules
- [ ] Watch bonus rounds multiple times if shown
- [ ] Capture screenshots of bonus features
- [ ] Update MEMORY.md with complete bonus feature details

**Deliverable:** Complete bonus features documentation.

---

**Deliverable:** Comprehensive CLEOPATRA slots documentation in MEMORY.md.

3. **Send Telegram Update:**
