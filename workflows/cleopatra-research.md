# cleopatra-research.md

This workflow is a **game-specific instance** of the 2-phase pipeline, scoped to Cleopatra or Cleopatra-style slots.

## Phase 1: Selection (Cleopatra Focus)

**Goal**: Find the best YouTube video for Cleopatra slot analysis.

### Search Targets

- Queries: "Cleopatra slot machine gameplay", "IGT Cleopatra bonus", "Cleopatra free spins"
- Priority: clear reels, visible paytable, multiple bonus rounds

### Output

- Top 5 Cleopatra videos sent to Ron for selection.

---

## Phase 2: Video Analysis (Cleopatra Focus)

**Goal**: Reverse-engineer Cleopatra math model and mechanics.

### Specific Items to Extract

- Reel layout (often 5x3)
- Symbols: Cleopatra, Pharaoh, Scarab, pyramid, card values
- Wilds/scatters: identify roles and triggers
- Bonus mechanics: free spins, multiplier behavior, retriggers
- Paytable values: capture from the paytable screen if shown

### Spreadsheet Expectations

Same 7-sheet format as the global workflow:

- game_overview
- symbol_inventory
- paytable
- math_model
- bonus_features
- visual_analysis
- analysis_log

### Delivery

- Send summary + spreadsheet via Telegram.

---

## Notes

- If multiple Cleopatra variants appear, note which version is analyzed.
- If RTP or volatility is not visible, mark as "estimated" and document source.
